Examples for Chapter 1
======================

These examples can be explored either by copying-and-pasting into the OCaml
top-level, or by using the prepared top level "examples.top":

```
$ ./examples.top 

     OCaml

# open Examples;;

# setify [1; 2; 3; 3; 2; 1];;
- : int list = [3; 2; 1]
```
